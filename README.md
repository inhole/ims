# IMS

Inventory Management System MVP backend assignment.

## Stack

- Java 17
- Spring Boot 3
- Spring Web
- Spring Validation
- Spring Data JPA
- PostgreSQL

## Build

This project is configured with Gradle.

```powershell
.\gradlew.bat test
.\gradlew.bat bootRun
```

## Database

The PostgreSQL DDL is available at [docs/schema.sql](docs/schema.sql).
