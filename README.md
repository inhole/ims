# IMS

Inventory Management System MVP backend assignment.

## Stack

- Java 17
- Spring Boot 3
- Spring Web
- Spring Validation
- Spring Data JPA
- PostgreSQL

## Build

This project is configured with Gradle.

```powershell
.\gradlew.bat test
.\gradlew.bat bootRun
```

The Gradle wrapper still needs to be generated or added before these commands can run in a clean checkout.
